- 由于外部厂商 API (如 MiniMax) 会出现偶尔输出异常嵌套对象导致前端崩溃（`[object Object]`）的现象，增强了 `normalizeReading` 的结构拍平逻辑，即使碰到 Object 也会将其强制转换为可读的 String，挽救前端 UI。

### 2.3 流式展示顺滑度体验优化

- 修复：使用了 `textLen` 获取剔除 `<think>` 内容后的纯文本长度再进行进度状态计算（"正在分析牌面" -> "正在解读关联..."）。
